KEYFRAME: Design Hub – User Guide
KEYFRAME is a lightweight event hosting and attendance platform designed specifically for the motion design and creative community. It streamlines the entire process from event creation to venue entry.

Core Platform Flows
1. Publish (For Organizers)

The journey begins with creating a digital home for your event.

Create Your Event: Navigate to the "Dashboard" and select "Create Event."

Define Categories: Fill in the required details, including the Technology (e.g., Motion Design, 3D) and Event Type (e.g., Workshop, Meetup). These fields will automatically generate a stylized cover for your event.

Go Live: Set your event status to "Published" to make it visible on the public Explore page for the community to discover.

2. RSVP (For Attendees)

Signing up is designed to be frictionless for the community.

Discover: Browse the Explore page to find upcoming events. You can filter by city or date to find what suits you.

Join the List: Click the RSVP button on any event page.

Waitlist Logic: If the event has reached its maximum capacity, the platform will automatically place you on the Waitlist. If a spot opens up, you will be moved to the confirmed list.

3. Ticket (Your Digital Pass)

Once your attendance is confirmed, the platform generates your entry credentials.

Access Tickets: Go to your personal "My Events" section.

Digital Pass: Every confirmed RSVP generates a unique Digital Ticket. This ticket contains a unique ID Code and a QR Code that acts as your entry pass for the venue.

4. Check-in (Venue Management)

Managing the door on the day of the event is simple and fast.

The Checker Portal: Organizers use a dedicated Checker Link accessible from their Dashboard.

Process Entry: Staff can either scan the attendee's QR code using the QR Scanner or manually enter the Ticket ID into the "Manual Check-in" field.

Real-time Tracking: The host dashboard updates in real-time, showing how many people have checked in versus total RSVPs.

Data Export: After the event, hosts can Export CSV data to analyze attendance and capture final outcomes.
